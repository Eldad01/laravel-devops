# Laravel DevOps Project

## Description

Projet DevOps complet utilisant Laravel, Docker, GitHub Actions et un VPS Ubuntu.

L’objectif du projet est de démontrer un pipeline CI/CD fonctionnel permettant :

- le build automatique
- les tests automatiques
- le push Docker Hub
- le déploiement automatique sur un VPS public

---

## Stack Technique

- Ubuntu Server 22.04
- Docker
- GitHub Actions
- Laravel 10
- SQLite
- VPS Contabo
- Docker Hub
- GitHub

---

## Fonctionnalités

- Déploiement automatique CI/CD
- Conteneurisation Docker
- Firewall UFW
- Déploiement VPS public
- Pipeline GitHub Actions
- Redémarrage automatique Docker

---

## Pipeline CI/CD

Le workflow GitHub Actions effectue automatiquement :

1. Tests Laravel
2. Build Docker
3. Push Docker Hub
4. Déploiement VPS

---

## Déploiement

L’application est déployée automatiquement sur un VPS Ubuntu via Docker.
